# Функционал загрузки резюме

## Описание

Бот теперь поддерживает загрузку резюме для улучшения качества генерации сопроводительных писем. ИИ анализирует ваше резюме и составляет персонализированные письма на основе вашего опыта и навыков.

## Как использовать

### 1. Загрузка резюме

**Вариант A: Загрузка файла**
- Отправьте боту файл с резюме (PDF, DOC, DOCX, TXT)
- Бот автоматически извлечет текст и сохранит резюме

**Вариант B: Текстовое сообщение**
- Отправьте текст вашего резюме боту обычным сообщением (длинное сообщение > 100 символов)
- Бот автоматически распознает это как резюме

### 2. Просмотр резюме

Используйте команду `/resume` для просмотра загруженного резюме.

### 3. Удаление резюме

Используйте команду `/clear_resume` для удаления сохраненного резюме.

### 4. Генерация письма с резюме

После загрузки резюме:
1. Получите вакансию от бота
2. Нажмите "✅ Да, составить"
3. ИИ автоматически использует информацию из вашего резюме для составления персонализированного письма

## Поддерживаемые форматы

- **PDF** - требуется библиотека PyPDF2
- **DOC/DOCX** - требуется библиотека python-docx
- **TXT** - поддерживается из коробки

## Установка дополнительных библиотек

```bash
pip install PyPDF2 python-docx
```

## Как работает ИИ-анализ

1. **С OpenAI API** (если установлен OPENAI_API_KEY):
   - ИИ анализирует описание вакансии
   - ИИ анализирует ваше резюме
   - ИИ составляет персонализированное письмо, подчеркивая релевантный опыт

2. **Без OpenAI API**:
   - Используется шаблонная генерация
   - Извлекаются ключевые слова из вакансии и резюме
   - Письмо составляется на основе найденных совпадений

## Хранение резюме

- Резюме сохраняются в директории `resumes/`
- Каждое резюме привязано к вашему Telegram user_id
- Резюме загружаются автоматически при перезапуске бота

## Безопасность

- Резюме хранятся локально на вашем сервере
- Каждый пользователь видит только свое резюме
- Резюме можно удалить командой `/clear_resume`
